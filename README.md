# 📂 Projects

## 🔐 File Encryption (128 / 256 bits)

A Python implementation of symmetric file encryption that demonstrates:

- Encryption using **128-bit and 256-bit keys**
- Password-based key derivation (**PBKDF2**)
- Random **salt** and **nonce**
- Stream cipher approach using **XOR**
- Data integrity verification using **HMAC-SHA256**
